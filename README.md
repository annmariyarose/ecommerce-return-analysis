# ecommerce-return-analysis
📊 Data analytics project exploring patterns in return behavior across e-commerce transactions. Includes visual insights, customer segmentation, and actionable trends.

# 📦 E-commerce Return Behavior Analysis using Python

This project dives deep into the **return patterns** of online shoppers through a synthetic dataset of e-commerce transactions. With the growing importance of minimizing returns in online retail, this analysis uncovers valuable insights into **who returns products**, **why**, and **when** — empowering businesses to make smarter logistical, pricing, and marketing decisions.

---

## 🔍 What This Project Covers

- 🛒 Product category return rates
- 🧍 Customer segmentation by age and gender
- ⏱️ Days to return analysis
- 💸 Product price distribution by payment method
- 📈 Return trends over time
- 🔄 Common reasons for returns
- 📉 Price vs return behavior
- 📌 Scatter plots, bar charts, box plots, heatmaps & more

---

## 📂 Dataset Used

- `ecommerce_returns_synthetic_data.csv`  
  A synthetic dataset containing order details, return status, customer demographics, pricing, and reasons for returns.

---

## 📊 Visualizations

- Bar plots for return rates by age group and product category
- Box plots for return delays
- Heatmaps for feature correlation
- Line plots for monthly return trends
- Pie chart of return reasons
- Scatter plot for return days vs product price
- Custom purple-themed color palette for aesthetic consistency

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

---

## 📎 Project Goals

- Understand key drivers of product returns
- Segment high-return demographics and products
- Present visual findings in a business-friendly and storytelling format

---

## 📌 Insights Example
